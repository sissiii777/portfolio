About Me

Chime of the Hidden Pulse

Physical Interaction Project 3

一个结合传统仪式（木鱼、祈祷）与现代物理计算和机器人技术的交互装置，通过生物识别数据（心率）生成响应式动作和声音。

交互逻辑

1

心率 < 60 BPM

10秒：非常缓慢，轻柔的敲击，如微风般
轻微单轴敲击（如Z轴上下运动）

transitionSmooth = 0.03;
amp1 = 70, amp2 = 30, amp3 = 20, amp4 = 50;
2

心率 60 ~ 70 BPM

8秒：逐渐加快节奏
多轴协调运动（如XY+Z拉伸敲击）

transitionSmooth = 0.05;
amp1 = 60, amp2 = 0, amp3 = 50, amp4 = 20;
3

心率 > 70 BPM

5秒：强烈、短促、密集的节拍
快速摆动并添加旋转轴
XYZ+Roll

transitionSmooth = 0.01;
amp1 = 50, amp2 = 40, amp3 = 50, amp4 = 40;

技术实现

MQTT
消息队列遥测传输协议，用于实时数据传输
BPM数据订阅
订阅心率数据字符串，实时获取用户生理状态
程序化机器人动作
根据心率数据控制机器人手臂的三种运动模式
多轴协调控制
XYZ轴及旋转轴的协调运动，实现不同强度的敲击动作
